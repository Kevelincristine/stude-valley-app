#🌾 StudeValley

> Transformando produtividade em uma experiência acolhedora inspirada em jogos de fazenda.

O **StudeValley** é uma aplicação web gamificada desenvolvida para transformar tarefas e estudos em uma jornada visual de evolução contínua. Inspirado na atmosfera de *Stardew Valley*, o projeto utiliza mecânicas de RPG, feedback visual em tempo real e uma identidade visual em pixel art para tornar a rotina mais envolvente e recompensadora.

Mais do que um simples gerenciador de tarefas, este projeto foi concebido como uma demonstração avançada de competências em desenvolvimento frontend moderno, arquitetura escalável de componentes, experiência do usuário e engenharia de software assistida por Inteligência Artificial.

# ⚙️ Principais Funcionalidades

## 🌾 Sistema de Plantio e Colheita

Cada tarefa criada representa uma “plantação” com diferentes níveis de dificuldade:

- 🥉 Cobre → 10 XP
- 🥈 Prata → 50 XP
- 🥇 Ouro → 100 XP

Ao concluir tarefas, o usuário acumula experiência e evolui sua fazenda digital.

---

## 🌱 Estufa Evolutiva

A estufa principal muda visualmente conforme o nível do jogador aumenta, utilizando GIFs e assets em pixel art renderizados com:

```css
image-rendering: pixelated;
````

---

## 📊 Dashboard Reativo

O sistema acompanha automaticamente:

* Taxa de eficiência
* Quantidade de tarefas ativas
* XP acumulado
* Troféus desbloqueados
* Nível do jogador

Tudo atualizado em tempo real através do estado global da aplicação.

---

## 🌦️ Sistema de Estações

O usuário pode alterar o clima da aplicação entre:

* Primavera 🌸
* Verão ☀️
* Outono 🍂
* Inverno ❄️

As mudanças são aplicadas dinamicamente através de variáveis CSS integradas ao Tailwind CSS.
