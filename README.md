# 🌱 Stude Valley

Um gerenciador de tarefas gamificado inspirado na estética de *Stardew Valley*. Transforme seus estudos em colheitas e evolua seu nível de conhecimento!

## 🚀 Status do Projeto
Este projeto está em desenvolvimento! Atualmente, concluí a **Semana 02 - Fase de Setup e Componentização**.

## 🛠️ Tecnologias Utilizadas
- **React** (com Vite)
- **TypeScript**
- **Tailwind CSS v4** (Estilização moderna)
- **Lucide React** (Ícones - *se for usar depois*)

## 🎨 Identidade Visual
As cores foram escolhidas para transmitir a paz e a organização de uma fazenda:
- **Primary:** Terracota (#cc9188)
- **Secondary:** Verde Sálvia (#839958)
- **Background:** Creme/Papel (#f7f4d5)
